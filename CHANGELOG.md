### Added
- Initial repository skeleton.
- Backend skeleton: Go + chi router, SQLite + golang-migrate, argon2id auth, JWT.

- CI workflows: golangci-lint, go test, dart analyze, flutter test, trivy image scan.
- Kustomize manifests targeting k3s on Raspberry Pi 5.
- Documentation: architecture, data model, API, vision pipeline, correlation, deployment, privacy, roadmap.
